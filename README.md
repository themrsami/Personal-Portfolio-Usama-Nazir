## ğŸš€ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/themrsami/Personal-Portfolio-Usama-Nazir.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Run development server**
   ```bash
   pnpm dev
   ```

4. **Build for production**
   ```bash
   pnpm build
   ```

5. **Start production server**
   ```bash
   pnpm start
   ```
